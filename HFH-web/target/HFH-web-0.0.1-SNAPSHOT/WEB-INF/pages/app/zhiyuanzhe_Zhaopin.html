志愿者招聘

我们招募这样的志愿者

1、十四周岁至七十周岁，身体健康，中国公民；

2、具有奉献精神，能够拿出一定的时间和精力用于消防志愿服务事业；

3、自愿加入火凤凰志愿者组织、遵守组织规章并在其中积极工作，能够参加消防培训、从事消防宣传，服从消防志愿者服务支队管理；

如果你满足以上条件，就申请加入吧！
