  厦门一泰消防科技开发有限公司成立于2005年，是一家以研发、生产、销售高效环保消防灭火剂，灭火器及消防安全设备系统的高新技术企业。一泰公司汇聚了一批专家团队，与厦门大学化工学院、中国科学院有机化学科研所、中科院城市公共安全技术研究中心、上海高新企业中科合臣股份有限公司等，建立了长期战略合作伙伴关系，确保公司的产品在国际、国内消防行业中，一直处于领先地位。

  一泰消防科技开发有限公司，是中国消防协会（CFPA）单位会员和美国消防协会（NFPA）单位会员；李江东董事长是全国消防标准化委员会第三分技术委员会委员。 公司自主研发生产的“一泰E-500”泡沫灭火剂集聚“技术领先、高效灭火、绿色环保、洁净无毒、广谱通用”五大优势。该产品获得中国国际专利技术与产品交易会金奖，并被国内外消防权威机构，评定为国际先进水平的消防产品。一泰公司已为国防军工、中国石油、中国石化、中海油、中化集团、SHELL、VOPAK、神华集团、福建高速、太古飞机维修等国内外知名企业，提供专业的消防产品与服务。 一泰公司的科研团队几经攻关，已成功自主研发公共汔车客舱固定灭火系统，即将为公交车驾乘人员的消防安全提供有力的保障。公司瞄准世界消防药剂技术的前沿，成功开发了E-500A类泡沫灭火剂（1%AFFF)、E-500高倍数泡沫灭火剂(3%G)，并将继续研究推进针对精密仪器、金属火等多种专业灭火剂及设备系统。 多年来，一泰人一直矢志不渝地耕耘着，把“提升消防安全”作为公司的宗旨；把“同创共享”作为公司的理念；把提供专业、全面、高效、贴心的消防安全产品与服务作为目标，努力为发展民族消防产业作贡献。公司计划未来在全国各省市设立销售网络和服务体系，并在全国选择几个重点区域建立生产基地，更好地为客户提供优质的消防安全产品和服务。
